# Airtable Hashtag Values

Predefined options for the `hashtags` Multiple select field in Airtable.

Admin assigns these during review — providers do not set them via Google Form.

---

| Group | Values |
| ------- | -------- |
| Language | `Speaks Ukrainian`, `Speaks English`, `Speaks Russian`, `Bilingual` |
| Service attributes | `Free consultation`, `Online / Remote`, `Mobile service`, `Accepts insurance`, `Licensed`, `Certified`, `Emergency available`, `Weekend available` |
| Texas cities | `Houston`, `Austin`, `Dallas`, `San Antonio`, `Fort Worth`, `DFW area` |

---

## Notes

- Use **Multiple select** field type in Airtable (not free text) to keep values consistent
- Add new options any time directly in Airtable without code changes
- These values are included in the site's search — users can find services by typing e.g. "Houston" or "online"
